# Medical Assistant Chatbot with TinyLlama

## Description

A notebook prototype for a medical-style assistant chatbot using Hugging Face Transformers and TinyLlama.

## Key Features

- TinyLlama text-generation pipeline
- Medical assistant prompt experiment
- Interactive notebook chat loop
- Runs in a Colab/Jupyter environment

## Tech Stack

- Python
- Jupyter Notebook
- Transformers
- TinyLlama
- PyTorch

## Installation

pip install transformers accelerate torch openai

## Usage

Open `medical_assistant.ipynb`, install dependencies, load the model, and use the interactive prompt loop.
